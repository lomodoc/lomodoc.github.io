              Lomorage客户端 #

              在运行了Lomorage服务程序后，您可以下载手机客户端，当前支持iPhone或者iPad。

              您也可以使用Web版本来上传电脑中已有的照片和视频，也可以在电脑上浏览已经备份的照片和视频。
