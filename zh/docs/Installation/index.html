                    安装说明 #

                    要使用Lomorage，您需要在本地网络运行"Lomorage照片助手"程序，您可以选择在Windows，macOS，Ubuntu，Raspbian, Armbain上运行，可以使用x86或者arm架构的机器。然后您需要去应用程序商店下载手机客户端应用。在运行了"Lomorage照片助手"程序之后，手机客户端会自动发现"Lomorage照片助手"，无需配置，然后您就可以开始备份照片和视频了。

                      1. 安装Lomorage客户端
